Eu sou

Sou técnico em informática há mais de 15 anos e atualmente estou em transição de carreira para a área de Engenharia de Software. Tenho me dedicado intensamente ao estudo de eletrônica, linguagens de programação, banco de dados e Internet das Coisas (IoT), com o objetivo de me aprofundar no desenvolvimento de aplicativos modernos e eficientes. Para reforçar meus conhecimentos, venho participando de formações como a graduação em Engenharia de Software, cursos técnicos e diversas plataformas de aprendizado online. Durante essa jornada, tive a oportunidade de desenvolver projetos utilizando tecnologias como SQL, Java, JavaScript e C#, sempre buscando explorar bibliotecas e frameworks com profundidade técnica. Sou entusiasta de boas práticas de desenvolvimento, como o uso de código limpo, testes automatizados, reaproveitamento de componentes e documentação clara. Além disso, procuro estar sempre atualizado com as tendências e tecnologias emergentes, mantendo um forte engajamento com a comunidade de desenvolvedores.

Meus Projetos

BR Sistemas – Gerenciamento de Bares e Restaurantes

Projeto de conclusão do curso Técnico em Desenvolvimento de Sistemas no Senac. Sistema desktop completo para gerenciamento de bares e restaurantes, incluindo cadastro de clientes, fornecedores, produtos, pedidos, formas de pagamento e geração de relatórios. Desenvolvido em Java com interface Swing, padrão MVC e persistência em banco de dados relacional.

Sistema de Orçamentos - Senai

Projeto final do curso técnico no Senai. Sistema desktop simples para geração de orçamentos comerciais, com estrutura de cálculo de produtos, totalização, controle de quantidades e preços. Desenvolvido com lógica estruturada, menus de console e abordagem didática.

Conversor Dólar - BACEN API

Conversor de moedas desenvolvido em PHP, que consome a API oficial PTAX do Banco Central do Brasil para obter a cotação do dólar. Inclui tratamento de datas não disponíveis (feriados/fins de semana), fallback automático, validação de entrada e exibição formatada dos valores. Utiliza cURL e JSON, com interface simples via HTML/CSS.
